# 그립컴퍼니 영화 검색 앱
https://movie-app-five-zeta.vercel.app/

- 영화 api를 호출하여 검색 후 즐겨찾기로 등록, 삭제 할 수 있는 React App

## 개발환경
- React 18.1.0
- node 18.0.0
- Typescript | React | SASS 

## 주요기능
- 검색
- 즐겨찾기
- 무한 스크롤
